# E-ARI typography

| Family | Use | License |
|---|---|---|
| **Plus Jakarta Sans** | Headings, hero titles, brand wordmark — weights 400 / 500 / 600 / 700 / 800 | Open Font License — Google Fonts |
| **Inter** | Body, paragraph, UI text — weights 300 / 400 / 500 / 600 / 700 | Open Font License — Google Fonts |
| **JetBrains Mono** | Eyebrows, labels, data, code — weights 400 / 500 | Apache 2.0 — Google Fonts |

## Conventions

- **Section eyebrows** are always JetBrains Mono · 10pt · uppercase · letter-spacing 0.22em · `#60a5fa` at 90% opacity, prefixed with a `8px × 1px` blue hairline rule.
- **Hero titles** use Plus Jakarta Sans 700 with an *italic* accent in `#60a5fa` 500 on the second clause — never on the first word.
- **Numerical data** (scores, KPIs, prices) uses `font-variant-numeric: tabular-nums` with Plus Jakarta Sans 600.
- **Mono labels** are always uppercase. Sentence case in mono is forbidden.
