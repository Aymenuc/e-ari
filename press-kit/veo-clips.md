# E-ARI — Veo clip breakdown

Veo generates **8-second clips maximum**. This file breaks both ads into individual Veo jobs.  
Stitch the outputs in order in your NLE (Premiere, DaVinci, CapCut). Each clip has:
- A **duration** (≤ 8 s — use that as the Veo length setting)
- An **image reference URL** to paste into Veo's "Reference image" field (or leave blank if none)
- A **Veo prompt** — copy verbatim

## Veo settings to use on every clip

| Setting | Value |
|---|---|
| Aspect ratio | 16:9 |
| Resolution | 1080p |
| Duration | match clip duration above |
| Camera motion | None (locked) unless the prompt says dolly/pan |
| Seed | lock seed once you find a consistent spokesperson look — reuse it across all clips |
| Reference image | paste URL from the clip header |

## Continuity tip

Generate CLIP 02 first (the spokesperson). **Note the seed.** Use that same seed for CLIP 03, 04, 06, 08, 09, 10 and all Part 2 clips that include the person. This keeps wardrobe, lighting, and face consistent across all clips.
